flights-app
===========

A simple web-based application that manages data about flights.

Completed as part of Dr. King-Ip Lin's CSI 3335 (Database) class at Baylor
University in the Spring 2015 semester.

Notable files:
- database.php -- contains database connection information
- flights-app.sql -- export of the database required to run the application
- index.html -- contains links to all pages
